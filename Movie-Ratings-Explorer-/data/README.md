# Data Directory

## Overview

This directory stores all data assets for the Movie Ratings Explorer project, organized by processing stage.

## Subdirectories

- **`raw/`**: Immutable source data (original CSV export from IMDb)
- **`processed/`**: Generated SQLite database and cleaned/transformed datasets

## Data Pipeline

1. **Raw CSV Import** (`raw/imdb_top_1000.csv`) → Pandas DataFrame
2. **Database Creation** → SQLite `.db` file in `processed/`
3. **Query Execution** → Results cached for visualization

## Storage Policy

- **Raw files**: Never modified after initial import
- **Processed files**: Can be regenerated by running the notebook
- **Total size**: ~2-3 MB (CSV + SQLite database)

## Accessing Data

All data operations in the notebook use portable `Path()` patterns:

```python
raw_data_path = Path('../data/raw/imdb_top_1000.csv')
db_path = Path('../data/processed/movies.db')
```

For manual access:
- **Inspect CSV**: Open in Excel or use `pd.read_csv()`
- **Query SQLite**: Use `sqlite3` CLI or any SQL client
  ```bash
  sqlite3 data/processed/movies.db
  SELECT COUNT(*) FROM Movies;
  ```
