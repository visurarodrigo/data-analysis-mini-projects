# Data Analysis Mini Projects

Professional monorepo of data analysis projects, each maintained in its own folder with independent assets, notebooks, and documentation.

## Overview

This repository consolidates multiple mini projects into one organized workspace while preserving the original development history of each project.

Each project now follows a cleaner structure and includes:

- project-specific README
- organized data, notebooks, and report assets
- explicit dependency file for reproducibility

## Projects

1. [medical-insurance-analysis](medical-insurance-analysis)
Focus: Regression modeling for insurance charge prediction and feature impact analysis.

2. [media-consumption-analysis](media-consumption-analysis)
Focus: Netflix content analysis with notebooks and an interactive Dash dashboard.

3. [Movie-Ratings-Explorer-](Movie-Ratings-Explorer-)
Focus: SQL + Python analysis of IMDb Top 1000 movies with SQLite-backed notebook workflows.

4. [-Stock-Price-Analysis-of-Apple-Inc](-Stock-Price-Analysis-of-Apple-Inc)
Focus: Apple stock trend, volatility, and return analysis using notebook-based time-series exploration.

## How To Use

1. Open any project folder from the list above.
2. Read the project README for details.
3. Create a virtual environment inside that project folder.
4. Install dependencies from that project requirements file.
5. Run the notebook or dashboard app for that project.

## Recommended Workflow

- Keep each mini project self-contained.
- Avoid mixing datasets or generated outputs between project folders.
- Track updates project-by-project with separate commits.
