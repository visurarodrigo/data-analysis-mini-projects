# app

This folder contains the production-facing Dash dashboard for the media consumption analysis project.

## Purpose
- Present key analysis outputs in an interactive web interface.
- Provide a lightweight presentation layer on top of project preprocessing utilities.

## Contents
- `dash_app.py`: Dashboard application entry point.
- `assets/`: Dash static assets (CSS and optional client-side files).
- `screenshot/`: Image previews of dashboard screens.

## Data Dependencies
`dash_app.py` expects:
- `../data/raw/netflix_titles.csv`
- `../notebooks/images/netflix_map.html` (generated by the interactive notebook)

If the map file is not available, the app shows an informative fallback message.

## Run
From the project root:

```bash
python app/dash_app.py
```

Open `http://127.0.0.1:8050` in your browser.
